# 🕹️ Transcendence (ft_transcendence_42)

Final project from the 42 Network (Common Core) — a real-time multiplayer Pong web app with social features.

---

## 🎯 Overview

Transcendence is a Single-Page Application (SPA) where users can:

- Play **multiplayer Pong** matches
- Chat in real time with other users
- Manage profiles, friends, leaderboards, and match history

---

## 🔧 My Contributions

- **Real-Time Chat System**: Designed and implemented WebSocket-powered private & public channels, along with moderation controls (mute/ban/block).
- **User Dashboard**: Built a responsive dashboard for profile settings, friend requests, stats overview, and match history.

---

## 🛠️ Tech Stack

| Frontend                 | Backend                | Real-Time        | Database    | Deployment |
|--------------------------|------------------------|------------------|-------------|------------|
| Next.js, React, TypeScript | Django (Django Channels) | WebSockets       | PostgreSQL  | Docker     |
| Tailwind CSS (or CSS)   | Django REST Framework  |                  | Redis  | Docker Compose |
